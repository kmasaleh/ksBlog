Angular Custom Modal Component

Published 26 October 2020 . 15 min read . 0 comments

Here we have a simple angular library named @ksaleh/modal that you may install it by issuing the npm i @ksaleh/modal from node command prompt. The library provides a simple custom modal component aka "Dialog Box" you may used it for multi purposes inside your code.

The modal component is a reusable widget that can be used for multiple purposes. It may display a simple informational message telling the user something like the application is about to delete somthing or displaying a feed back from a server process. You will desgin the template that is used inside the modal by yourself so you may tell if it has a draggable caption, contain some icon or picture or even buttons and inputs. You will supply all of that information through a congiuration object that is registered by using a template registeration service along with an instance of a data provider context object that is used to bind with the template elements with data. After that you can hook your methods for intercepting the modal events which will be triggered during execution and upon exiting from the modal.
